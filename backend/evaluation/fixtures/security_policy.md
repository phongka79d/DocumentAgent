# Security Policy

## Access Review

Privileged workspace access is reviewed every twenty-one days. The review proof phrase is LANTERN-21.

## Incident Response

Critical incidents must be acknowledged within eleven minutes and assigned the code VIOLET-11.

## Billing Archive Exception

Security operations retain incident-linked billing exports for sixty days. The exception marker is SILVER-60.

## Policy Link

Account-access refund investigations are jointly governed with the Pricing Policy.
